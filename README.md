# Учебная конфигурация 1С: Управление заказами яхт

Учебный проект по курсу «Разработчик 1С» в ТГУ.

---

## Описание проекта

Конфигурация предназначена для автоматизации управления заказами на яхты и сопутствующие товары. Реализован полный цикл: от ведения справочников до оформления заказов, заключения договоров и отчетности.

---

## Функциональные возможности

**Справочники:**
- Клиенты
- Контактные лица (ФИО, дата рождения, паспорт, телефон, email, адрес)
- Продукция (тип лодки, количество мест, наличие мачты, цвет, материал, налог)
- Аксессуары (описание, налог)
- Сотрудники (телефон, тип сотрудника)
- Организации (ИНН, КПП, адрес)
- Поставщики
- Типы яхт
- Цвета
- Материалы

**Документы:**
- Заказ (клиент, менеджер, адрес доставки, дата доставки, сумма, спецификация)
- Договор (клиент, заказ-основание, сумма, дата подписания, статус, график платежей)

**Отчеты (СКД):**
- Частые клиенты
- Самые дорогие заказы

**Регистры:**
- Цены (номенклатура, цена)

**Роли:**
- Администратор
- Старший менеджер
- Менеджер

**Подсистемы:**
- Управление
- Поставщики
- Продажи

**Прочее:**
- Перечисление "Тип сотрудника"
- Общая форма "ЯхтСтрой" (начальная страница)
- Печатная форма договора
- Общие картинки для подсистем

## Как запустить

1. Установите учебную версию 1С:Предприятие 8.3.
2. Создайте новую информационную базу.
3. Загрузите ProjecInternshipt.dt из папки dist/ через Конфигуратор → «Загрузить информационную базу».
4. Запустите 1С:Предприятие в режиме "Управляемое приложение".
